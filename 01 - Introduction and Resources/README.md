# Free HTML & CSS Learning Resource (Beginner Friendly)

Welcome to this **free learning initiative** for HTML & CSS!
This guide is for **absolute beginners** who want to start web development. It contains everything you need. From tools setup, learning resources, projects, quizzes, books and practice questions.

---

## Tools & Setup

Before starting, you will need:

- **[VS Code](https://code.visualstudio.com/)** – Recommended code editor.
- **[Live Server Extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)** – Auto-refresh browser on file save.
- **Web Browser** – Chrome, Firefox, or Edge (with DevTools).
- **[Git & GitHub](https://git-scm.com/)** – For version control & hosting projects.
- **[CodePen](https://codepen.io/)** / **[JSFiddle](https://jsfiddle.net/)** – Online playground for HTML & CSS.
- **[MDN Web Docs](https://developer.mozilla.org/)** – The official reference for HTML & CSS.
