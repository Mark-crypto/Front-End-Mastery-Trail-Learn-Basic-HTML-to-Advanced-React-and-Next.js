# Contributing to Frontend Journey

First off, thank you for considering contributing. Your help makes this free learning journey possible for thousands of learners around the world.

We want this to be a **welcoming and collaborative space** where contributions big or small are valued.

---

## Ways to Contribute

There are many ways you can help:

- Improve explanations or fix typos in lessons
- Add more examples, challenges or projects
- Translate content into other languages
- Create demos or interactive snippets
- Suggest resources, tutorials or tools

---

## Getting Started

1. **Fork the repo**

   ```bash
   git fork https://github.com/Mark-crypto/Front-End-Mastery-Trail-Learn-Basic-HTML-to-Advanced-React-and-Next.js
   ```

2. **Clone your fork**

   ```bash
   git clone https://github.com/Mark-crypto/Front-End-Mastery-Trail-Learn-Basic-HTML-to-Advanced-React-and-Next.js
   cd Front-End-Mastery-Trail-Learn-Basic-HTML-to-Advanced-React-and-Next.js
   ```

3. **Create a branch**

   ```bash
   git checkout -b feature/amazing-contribution
   ```

4. **Make your changes**

---

## Submitting a Contribution

1. **Commit your changes** with a descriptive message:

   ```bash
   git commit -m "Add: new example for CSS selectors"
   ```

2. **Push your branch**

   ```bash
   git push origin feature/amazing-contribution
   ```

3. **Open a Pull Request**

   - Go to the original repository
   - Click **New Pull Request**
   - Describe what you’ve added/changed and why

We’ll review it together .

---

## Style Guidelines

- Keep lessons **beginner-friendly** (explain jargon simply).
- Use **consistent formatting** for code (follow Prettier/ESLint if applied).
- Keep commit messages **short and descriptive**.
- For markdown:

- Use headers (`#`, `##`) for sections
- Use code fences (```) for code
- Use lists and tables where helpful

---

## Need Help?

- Open an [issue](issues) if you’re unsure about your idea.
- Join discussions in the repo (coming soon: community chat).

We are excited to learn and build together!
